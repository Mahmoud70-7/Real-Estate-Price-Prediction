# Real Estate Price Prediction Platform

An end-to-end data science platform designed to estimate real estate prices and provide market intelligence insights to support investment decisions.

---

## Business Problem
Real estate agencies and investors often face challenges in accurately pricing properties and identifying profitable investment opportunities.  
This project aims to solve this problem by combining **machine learning price prediction** with **interactive market intelligence dashboards**.

---

## Project Overview
The project delivers a complete analytics pipeline starting from raw real estate data to predictive models and visual dashboards.  
It integrates data preprocessing, regression modeling, experiment tracking, and business-focused visualizations.

---

## Key Deliverables
- Cleaned and fully processed real estate dataset  
- Feature engineering and location-based insights  
- Machine learning regression model for price prediction  
- Model experimentation and tracking using **MLflow**  
- Interactive **Power BI dashboards** for market intelligence and ROI analysis  
- **Streamlit-based property valuation application** (local)

---

## Tech Stack
- **Python** (Pandas, NumPy, Scikit-learn)
- **Machine Learning:** Linear Regression
- **MLflow** (model tracking and comparison)
- **Power BI** (market and investment dashboards)
- **Streamlit** (property valuation app)
- **PowerPoint** (final investor presentation)
